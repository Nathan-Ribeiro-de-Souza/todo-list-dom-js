# To-Do List (DOM)

A simple interactive to-do list application built using HTML, CSS, and JavaScript, focused on DOM manipulation and front-end fundamentals.

## About the Project

This project is a task management interface that allows users to add and remove tasks dynamically. It was developed to practice JavaScript DOM manipulation, event handling, and UI updates.

## Technologies

* HTML5
* CSS3
* JavaScript (DOM)

## Features

* Add new tasks
* Remove tasks
* Dynamic DOM updates
* Basic responsive layout

## Project Structure

```
/ (root)
  index.html
  style.css
  script.js
```

## What I Learned

* DOM selection and manipulation
* Event handling in JavaScript
* Updating UI dynamically based on user interaction
* Structuring interactive front-end applications

## How to Run

Open the `index.html` file in a web browser.
